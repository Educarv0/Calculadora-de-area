Calculadora de Área de Figuras Geométricas

Triângulo

O triângulo é uma figura geométrica com três lados. A área é calculada pela fórmula:

Área = (base × altura) / 2

Quadrado

O quadrado é uma figura geométrica com quatro lados iguais. A área é calculada pela fórmula:

Área = lado × lado

Retângulo

O retângulo é uma figura geométrica com quatro lados, onde os lados opostos são iguais. A área é calculada pela fórmula:

Área = largura × altura
